= Chapter 2: 작업 흐름 Git 명령어

Git은 초기 설정 및 시작, 작업 흐름, 브랜치 작업, 원격 저장소 동기화를 위한 명령어들을 제공합니다. 통합 개발 환경을 사용하고 있더라도 터미널 명령어를 알고 있으면 GUI 도구가 해결해주지 못하는 복잡한 상황을 제어하거나 문제가 생겼을 때 해결하는 방법을 찾을 수 있습니다.

이 단원에서는 작업 흐름에 쓰이는 git 명령인 status, add, commit, log 명령을 알아봅니다.
